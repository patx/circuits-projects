pastebin about

"circuits-pastebin" is an extremly simple, absolutely no frills, fast pastebin application.

it is written, as the name implies, using circuits.web.

pastes are highlighted using pygments.

there is no minumun or maximum garuntee of how long a paste will be available.

all pastes are open to the public.

if you would like a paste removed please contact harrison.

pastes are stored using mongodb, mlab, and mongoengine.

you can see the source code for this application here.
